  💡 CUSTOMIZATION
═══════════════════════════════════════════════════════════════

WANT DIFFERENT ICONS?
   1. Create/download 24x24 or 32x32 PNG images
   2. Name them to match existing files
   3. Replace in assets/ folder
   4. Restart GUI

WANT CUSTOM LOGO?
   1. Get Fun Run 4 icon (or any image)
   2. Resize to 128x128 pixels
   3. Save as assets/logo_128.png
   4. Restart GUI

REGENERATE EVERYTHING:
   rmdir /s /q assets
   python download_icons.py
